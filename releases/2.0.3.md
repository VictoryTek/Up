## What's Changed

### Improvements
- **Update progress bar**: A progress bar now fills beneath the "Updating…" status label as each package manager completes, giving clear visual feedback during multi-backend updates.
- **Responsive layout**: The update page now scales properly at larger window sizes and fullscreen. The content area widens, status labels no longer truncate early, and the terminal output panel fills all available vertical space when expanded.

### Bug Fixes
- **Flatpak**: Fixed update check always reporting no updates available when Flatpak apps are installed system-wide — the check now queries both user and system installations, matching what the update command does.
