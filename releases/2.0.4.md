## What's Changed

### Improvements
- **Update button**: Moved to a fixed footer so it remains accessible regardless of content height.
- **Cancel action**: Added a Cancel button to allow aborting an in-progress update run.

### Bug Fixes
- **NixOS**: Fixed a false positive where the NixOS backend incorrectly reported updates available when there were none.
- **VexOS**: Fixed a false positive in VexOS update detection and corrected the VexOS update command check.
- **Flatpak**: Fixed a false negative where available Flatpak updates were not detected.
