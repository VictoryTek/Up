# Up 1.0.2

**Patch release** fixing Determinate Nix upgrades and Nix profile compatibility with modern Nix versions.

## Bug Fixes

- **Fixed Determinate Nix upgrade failing with "requires root privileges"** — `determinate-nixd upgrade` is now routed through `pkexec` so it runs with the necessary privileges; `needs_root()` is also correctly set for Determinate Nix installations
- **Fixed Nix profile upgrades not pulling updates after the first run** — `nix profile upgrade .*` (the legacy regex syntax) is no longer recognised by Nix ≥ 2.18; Up now tries `nix profile upgrade --all` first and falls back to `.*` for older Nix installations
