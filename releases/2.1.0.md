## What's Changed

### New Features
- **VexOS cache-block dialog**: When a VexOS update is paused because kernel packages require a local source build the binary cache hasn't finished yet, Up now shows a dialog explaining exactly what's blocked and lets you choose `just deploy` (apply pending config without bumping nixpkgs), `just update-all` (force the local build now), or Wait (closes Up to retry once Hydra catches up).
- **VexOS brand theme**: New cyan/orange visual theme — navigation tabs moved into the header bar, a hero area with app icon and live status, and a themed Update All button, progress bar, and banners.

### Improvements
- **Layout**: Update All and Cancel buttons moved into the hero row, saving a full row of vertical space; the terminal output panel is capped at 200px when expanded so it no longer crowds the content above it.
- **Terminal panel**: Now expanded by default on launch so update output is immediately visible; default window height increased to accommodate it comfortably.
