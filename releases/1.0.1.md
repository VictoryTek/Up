# Up 1.0.1

**Patch release** fixing backend update sequencing and Nix terminal output.

## Bug Fixes

- **Fixed Nix and Flatpak running simultaneously** — backends now execute strictly one at a time; Flatpak no longer starts before Nix has finished
- **Restored Nix terminal output** — NixOS rebuild commands are now prefixed with `stdbuf -oL -eL` to force line-buffered output through the non-TTY pipe, so log lines stream in real time instead of appearing all at once at the end
- **Added `--print-build-logs` to `nixos-rebuild switch`** — build log lines are now emitted and visible in the terminal panel during a NixOS rebuild
