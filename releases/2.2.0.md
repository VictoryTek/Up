## What's Changed

### Improvements
- **Package list popover**: Viewing a backend's updated packages no longer expands the row inline and pushes the whole page into a scrollbar. Each source row now shows a package-count button that opens a small popover with the full list, so the main window never grows or scrolls to show it.
- **Accurate updated-items list**: The list of updated packages shown after a run now reflects what was actually updated, instead of a placeholder or mismatched set of items.
